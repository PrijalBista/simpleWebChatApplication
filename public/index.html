About This Application

This is an simple group chat application where no logs of the information are stored in server or database hence stateless. All the chat information stays only until browser refreshes and thus chat history cannot be accessed on later state.

This application can be used for group discussion about things which donot matter later (no need to see the history later ) and hence can be used for anonymous chatting (self destructable)

You have to use a nickname initially to start group chatting

|| NOTE || : No data is stored in the server. So all data is lost after you refresh your browser :).No server side caching no client side caching as well :)
